## Install
```
$ npm install --global omfg
```


## Usage

```
$ omfg --help

  Usage
    $ omfg [<directory|glob> ...]

  Options
    --init          Initialized immediately with only README file
    --license       Choose the license
    --ignore        Set useful .gitignore files
    --github        Use GitHub templates

  Examples
    $ omfg
    $ omfg --ignore node
    $ omfg --license mit
    $ omfg --github
```
